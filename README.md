# Privacy Policy for QuickCompress

QuickCompress respects your privacy.

This app works completely offline and does not collect, store, or share any personal data.

## Information Collection
QuickCompress does not collect any personal information.  
All images selected by the user remain on the device and are never uploaded to any server.

## Permissions
The app requests access to photos/media storage only to allow users to select images for compression.  
This permission is used solely for app functionality.

## Data Sharing
QuickCompress does not share any data with third parties.

## Internet Usage
The app does not require an internet connection for image compression.

## Changes
If this privacy policy changes in the future, updates will be reflected in the app listing.

## Contact
If you have questions about this privacy policy, you may contact the developer via the email listed on the Play Store.
